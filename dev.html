ЭЛЕМЕР-ИКСУ-2012




ЭЛЕМЕР-ИКСУ-2012 предназначен для воспроизведения и измерения:

  • Электрических сигналов силы постоянного тока;
  • Электрических сигналов напряжения постоянного тока;
  • Электрических сигналов сопротивления постоянному току;
  • Сигналов термометров сопротивления по ГОСТ 6651-2009, DIN 43760;;
  • Сигналов преобразователей термоэлектрических по ГОСТ Р 8.585-2001;
  • Сигналов приборов, поддерживающих HART-протокол*;
  • Сигналов преобразователей давления эталонных ПДЭ-020(И)*;
  • Сигналов термометра цифрового эталонного ТЦЭ-005/М3*.

Мультиметр Fluke




Мультиметр Fluke 1587 – компактное устройство, которым действительно
удобно пользоваться. Прибор применяется для выполнения широкого спектра
измерений. Он порадует высокой точностью, полученные результаты можно
полноценно использовать.

Мультиметр Fluke 1587 поставляется с кейсом. В нем можно транспортировать
прибор и хранить. В комплекте предоставляется дополнительная оснастка,
которая пригодится при выполнении измерений.
